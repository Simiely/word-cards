# Word Cards — 儿童双语学习卡片

一个面向学龄前及小学低年级儿童的交互式双语学习卡片项目，帮助孩子在趣味中学习中文和英文词汇。

## 整体规划

```
首页 (index.html)
├── 🦁 动物卡片     ← 当前开发中
├── 🍎 水果卡片     (待规划)
├── 🚗 交通工具     (待规划)
├── 🌈 颜色形状     (待规划)
└── ...更多类别
```

### 当前进度：动物卡片

📍 在线预览：[https://simiely.github.io/word-cards/animal-cards.html](https://simiely.github.io/word-cards/animal-cards.html)

#### 两种学习模式

| 模式 | 页面 | 说明 |
|------|------|------|
| **卡片模式** | `animal-cards.html` | 随机排序，顺序翻阅，每次一组预加载 |
| **浏览模式** | `animal-select.html` | 瀑布流方阵，自由选择，点击弹出卡片详情 |

#### 功能清单

- **21 种常见动物**：狮子、大象、熊猫、老虎、长颈鹿、斑马、企鹅、海豚、猴子、兔子、猫、狗、马、牛、羊、鸡、鸭子、鱼、蝴蝶、青蛙、鲨鱼
- **双语名称**：中文 + 英文，点击朗读发音（预录制 MP3，全平台兼容含 iOS Safari）
- **科普描述**：每种动物配有中文科普介绍，点击朗读
- **实拍照片**：每种动物配一张真实照片，支持缩放和拖拽
- **图片交互**：
  - 缩放模式切换按钮（右下角）
  - 鼠标滚轮缩放（以光标为锚点）
  - 双指缩放（移动端）
  - 自由拖拽平移
  - 边缘平均色背景（图片加载后自动提取）
- **分组预加载**：21 个动物随机排序后按 6 个一组切分，首屏预加载 3 组，翻页零等待
- **浏览模式**：瀑布流方阵（自适应 2/3/4 列），emoji 平均色方块背景，已浏览动物灰度标识
- **加载动画**：呼吸缩放 emoji，加载完成后自动淡出
- **循环导航**：上一个 / 下一个 / 随机，键盘左右箭头翻页、空格随机
- **竖屏优先**：响应式设计，移动端优化

#### 文件结构

```
animal-cards.html          ← 卡片模式页面
animal-select.html         ← 浏览模式页面
animal/
├── images/                ← 动物图片（{英文名}.jpg，21 张）
├── speech_zh/             ← 中文名称语音（Edge TTS，21 个）
├── speech_en/             ← 英文名称语音（Edge TTS，21 个）
└── speech_fact/           ← 科普描述语音（Edge TTS，21 个）
```

## 技术栈

- 纯 HTML/CSS/JS，无框架依赖
- Edge TTS — 预录制 MP3 语音（全平台兼容，包括 iOS Safari）
- Canvas API — 图片边缘色提取、emoji 平均色提取
- GitHub Pages — 静态托管
- 响应式设计 — 移动端优先
- localStorage — 浏览记录持久化

## 本地运行

直接浏览器打开 HTML 文件，或使用任意 HTTP 服务器：

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080/animal-cards.html
# 或   http://localhost:8080/animal-select.html
```
